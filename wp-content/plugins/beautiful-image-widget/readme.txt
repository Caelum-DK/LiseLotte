== Description ==

Beautiful image widget is a WordPress plugin that allows you to display random image from huge high-quality images collection on the sidebar of your blog. Supports multiple images. 

= Requirements =

* WordPress 3.0+
* PHP 5.1+
* At least 160 pixels sidebar width

== Installation ==

= Install =

1. Download the plugin
1. Unzip the 'beautiful-image-widget.zip' file. 
1. Upload the the 'beautiful-image-widget' folder to your 'wp-contents/plugins' folder. 

= Activate =

Once you completed installation process described above, perform the following steps:

1. Log into your WordPress administration panel and go to 'Plugins' page
1. Activate 'Beautiful Image Widget' plugin and proceed to the 'Appearance' > 'Widgets' page
1. Drag 'Beautiful Image' widget into the designated widget area and drop it to the slot of your choice

= Configure =

Beautiful image widget has three configuration options: 

1. Title - if you wish to override the default 'Beautiful Image' widget title, you may type your own title in that field
1. Number of pictures - you may choose how many pictures would be displayed on the sidebar. At the time you may display up to three photo thumbnails
1. Display link - if you wish to support our project development, please, tick this checkbox. If you leave this checkbox checked, a small link would be printed right below the photos

If you'll find any bugs or have any suggestions, please, let us know.
